配置Github

1.注册不多说
2.新建一个仓库，名为username.github.io

3.并添加ssh公钥在github的setting里

番外-如何获得ssh公钥(第一次使用的话)

1.在桌面右键 Git Bash 键入：

2.生成过程中会让你输入passphrase口令，可以直接回车，以后出现都是空密码
3.生成密钥后，在目录找到私钥 id_rsa 和公钥 id_rsa.pub 这两个文件。id_rsa.pub 里的就是公钥，复制里面的内容就行。
PS：千万不要把私钥文件 id_rsa 透露给任何人。否则别人可以以管理员身份登录你的blog。
